和文件相关的快捷键，一般以 `C-x` 开始


## 文件创建 {#文件创建}

| 快捷键      | 含义          | 助记  |
|----------|-------------|-----|
| `C-x` `C-f` | 查找文件，不存在则新建文件 | find  |
| `C-x` `C-w` | 另存为新文件  | write |
|             |               |       |
