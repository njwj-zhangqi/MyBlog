## 安装Vue CLI
```bash
pnpm install -g @vue/cli
```

## 创建Vue项目
```bash
vue create my-vue-app
```
按照需要，手动选择需要的配置项

## 安装项目依赖
```bash
cd my-vue-app
pnpm install
```

## 启动服务
```bash
pnpm run serve
```
